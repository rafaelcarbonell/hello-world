Welcome to VET Surgeon!

Welcome to our new application from Media Interactiva, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

To help you get started, we've some indications to help you:

  • Home. Current page where you can have an overview about the application.
  • Employees. In the Employees section we have a list of registered employees that want to use the vet application.
  • Pets. In the Employees section we have a list of registered pets owned by our employees that want to use the vet application.
